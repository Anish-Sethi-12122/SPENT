# Nothing Brand Fonts

This app uses Nothing's proprietary typography. The font files are **not included** in this repository due to licensing restrictions.

## Required Font Files

Place the following files in this directory:

| File | Font | Source |
|------|------|--------|
| `ndot.otf` | Ndot 57 | [Colophon Foundry](https://www.colophon-foundry.org/) — designed for Nothing |
| `Inter.ttf` | Inter | [Google Fonts](https://fonts.google.com/specimen/Inter) (free, open source) |
| `NType82.ttf` | NType 82 Mono | Designed exclusively for Nothing (2021) |

## How to Obtain

### Inter (Free)
Download from [Google Fonts](https://fonts.google.com/specimen/Inter) and place `Inter-Medium.ttf` here, renamed to `Inter.ttf`.

### ndot & NType82 (Proprietary)
These are proprietary fonts owned by Nothing Technology, created by Colophon Foundry. Options:
1. If you participated in the Nothing hackathon, you may have received these fonts as part of the design kit
2. Extract from a Nothing Phone device (for personal use only)
3. Use the open-source [NDOT 45](https://fontstruct.com/fontstructions/show/2209196/ndot-45-inspired-by-nothing) alternative (inspired by Nothing, slightly different appearance)

## Without These Fonts

The app will still **run** without the font files — it will fall back to system default fonts. The layout and functionality are identical, only the typography will differ from the original Nothing design.
